# Fake News Detection System

This project detects whether a news article is **real or fake** using
Machine Learning and Natural Language Processing techniques.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- NLTK

## Model Used
- TF-IDF Vectorizer
- Logistic Regression

## Platform
- Google Colab

## How to Run
Open the notebook and run all cells sequentially.
